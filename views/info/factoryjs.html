Factory JS

Defining a Factory/Service
  • angular.module('modulename').factory('FactoryName', function('dependencies(un-ordered)'){ code return factoryinstance/some object/funtion })
  • modulename.factory('FactoryName', function('dependencies(un-ordered)'){ code; return factoryinstance/some object/funtion; })
Our application we have a ComponentController.js
demo.factory('notify', function ($window) {
var msgs = [];
return function (msg) {
msgs.push(msg);
if (msgs.length === 3) {
$window.alert(msgs.join("\n"));
msgs = [];
}
};
});
