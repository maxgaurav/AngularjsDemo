Forms

  • Basic
  • Example
  • Angular provides powerful means to control forms
  • Forms in angular provide validation indication
  • Inputs in forms can be made required dynamically using ng-required
  • You can use the ng-submit attribute to call function to send the form
  • The <ng-form> to bring all the functionalities of form tag
  • Following css class are added to input according to form state
    • ng-valid: the model is valid
    • ng-invalid: the model is invalid
    • ng-pristine: the control hasn't been interacted with yet
    • ng-dirty: the control has been interacted with
    • ng-touched: the control has been blurred
    • ng-untouched: the control hasn't been blurred
