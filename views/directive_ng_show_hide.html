Directives ng-show and ng-hide

  • The ngShow and ngHide directive's shows or hides the given HTML element based on the expression provided to the attribute.
  • The element is shown or hidden by removing or adding the .ng-hide CSS class onto the element. The .ng-hide CSS class is predefined in AngularJS and sets the display style to none (using an !important flag)
  • ngShow and ngHide expects their valu to be true or false
